# jumbo-uptime

provides a big chunky (jumbo) window for users if uptime is unsatisfactory. Hence the name jumbo.

variables are:
- LOGOPATH path for logo
- limit for time limit
- instructs eg. 'Send a fax to the support desk at 800-555-1212'
